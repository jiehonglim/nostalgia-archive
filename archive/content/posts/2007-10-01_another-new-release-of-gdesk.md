# Another new release of GDesk

[GDesk has a new version release 0.14](http://gdesk.wetpaint.com/page/GDesk+versions).

> 

v0.14 – 30th September 2007

Warning – this version has a major change and needs plenty of testing. If you don’t want to risk potentially large bugs and problems it might be best to wait a few days to see what comes up!

Either way it’s a good idea to keep v0.13 installs handy and back up your design before installing v0.14!

Note – this version introduces a slight change to the plug-in interface code so you’ll need to install new versions of plug-ins!

As the warning above, “Proceed with caution !!”
