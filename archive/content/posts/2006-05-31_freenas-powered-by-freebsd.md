# FreeNAS, powered by FreeBSD

[FreeNAS](http://www.freenas.org/) is a free NAS server. Install on any x86 pc and convert it to a NAS serving CIFS (samba), FTP, NFS, RSYNC protocols.

Software RAID is supported at the following levels, 0, 1, 5. All configuration is done through the web interface.

Maybe I can put my old dual P3 system to good use.
