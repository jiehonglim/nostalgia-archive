# Free & Useful HTC Touch Pro Applications

Listing some of the free & useful apps that I used for my HTC Touch Pro.

–

[PHM Registry Editor](http://www.phm.lu/Products/PocketPC/RegEdit/) – alternative registry editor, I was able to Import/Export with this editor.

–

[Gyrator](http://gyrator.brins0.net/) – a very flexible tool to perform various actions based on certain events, like screen orientation or stylus insertion or removal.

–

[PocketPuTTY](http://www.pocketputty.net) – everyone’s favourite tool, Secure Shell Client

–

[Klaxon](http://www.koushikdutta.com/search/label/Klaxon) – alternative alarm clock

Basically it’s a just an Alarm Clock, however you can snooze or off the alarms with movements. E.g flipping your phone to snooze and shakig it with 20G of force to off it.

–

[Advanced Configuration Tool](http://www.touchxperience.com/en/advanced-configuration-tool.html) – System Tweaks

There are over 200 settings covering user interface, performances, power management and HTC Touch Flo configurable.

–

[Advanced Task Manager](http://www.touchxperience.com/en/advanced-task-manager.html) – System Monitoring

A better task manager for  system monitoring, list and closes active applications and displays usage of storage space.

–

[Registry Editor](http://www.touchxperience.com/en/registry-editor.html) – System Tweaks

As it is, a Windows Mobile registry editor.

–

Finally to note, some applications requires the [Microsoft .NET Compact Framework 3.5](http://www.microsoft.com/Downloads/details.aspx?familyid=E3821449-3C6B-42F1-9FD9-0041345B3385&displaylang=en)
